# Guia de Implantação e infraestrutura

## pré-requisitos de ambiente(local)
Para rodar este projeto no seu computador você precisa ter instalado:
- [ ] PHp (versão 8.2);
- [ ] Composer (gerenciador de pacotes PHP)
- [ ] Filament (framework para o laravel)
- [ ] NPM
- [ ] Banco de Dados MySQL

## Checklist de instalação
- [ ] Criar estrutura de base de Dados no Laravel
- [ ] Instalar o filament
- [ ] Configurar o arquivo de variáveis de ambiente('.env').
- [ ] Rodar as aplicaçãoes de banco de dados('migrations').
