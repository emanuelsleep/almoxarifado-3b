# requisitos funcionais do sistema 

- Funcionalidades que o sistema deve ter:

## cadastro de produto
- nome
- marca
- estoque

## movimentos de entrada
- nome do produto
- quantidade de produto
- tipo

## movimento da saída
- consulta de estoque
- quantidade de saída
- histórico

# requisitos não funcionais 
- funcionamento em diversos aparelhos eletrônicos
- interface automatizada
# regras de negocio
- não permitir saldo negativo
- não permitir saída de produto sem estoque
